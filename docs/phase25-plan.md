# Phase 25 — V6 category-defining platform

V6 moves Anderoute into a category-defining enterprise logistics
operating network. We track evidence and maturity; we do NOT claim IPO,
acquisition, SOC 2, ISO, CarPlay or Android Auto as complete.

## Non-goals
- Fully autonomous dispatch
- IPO / acquisition / SOC 2 / ISO / mobile certification claims as complete
- Autonomous vehicle workflows, global customs, international localization,
  insurance underwriting, fully automated legal compliance.

## Exit criteria
- All routes render with mock data and clearly labeled placeholders.
- Schema, RLS, Edge Function plan committed.
- V6 demo flow runs end-to-end.
- Phase 26 plan committed.
