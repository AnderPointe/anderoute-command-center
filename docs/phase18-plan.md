# Phase 18 — Enterprise V2.5 plan

V2.5 hardens V2 for enterprise customers.

## In scope
- Production EDI (204/990/214/210/997 certified with 3+ trading partners)
- API monetization (metered usage, tiered rate limits, billable API plans)
- Advanced optimization (multi-stop, multi-constraint, hazmat + HOS aware)
- Advanced AI (model training feedback loop, supervised recommendation acceptance signal)
- White-label custom domains for customer portal + driver app
- Enterprise security controls (SCIM, advanced SSO, IP allowlists, session policies)
- Larger fleet scaling (500+ active drivers per company, sharded GPS streams)
- Production SOC 2 evidence collection (still not full automation)

## Deferred to V3
- Fully autonomous dispatch
- Carrier marketplace
- Native turn-by-turn / Android Auto / CarPlay (re-evaluate after V2.5)
- Full ML pipeline with retraining cadence
